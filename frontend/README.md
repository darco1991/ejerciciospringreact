# Frontend react para la integracion con spring


### Configurar
.env
```
PORT=8081
```

### Instalacion

En la carpeta del proyecto ejecutar:

```
npm install
# or
yarn install
```

### Compilacion y hot-reloads para desarrollo

```
npm start
# or
yarn start
```

Abrir [http://localhost:8081](http://localhost:8081) para visualizar los cambios
